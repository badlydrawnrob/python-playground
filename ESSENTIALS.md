# Python essentials

| *Python symbol*         | *Description* |
| ----------------------- | ------------- |
| `print(f"text {var}")`  | Print some text |
| `"""`                   | Text blocks or comments |
| `+=`, `-=`		  | Add or minus from variable |
| `;`                     | Allow multiple variables on one line |
| `true`, `false`         | Boolean values are lowercase |
| `return value`          | Most functions must return something |
| `not`, `and`, `or`      | Operators are lowercase |
| `&`, `|`                | Alternative syntax for `and`/`or` |
| `upper()`, `.upper()`   | Functions often have functions _and_ methods |
| `if [expression]:       | Does not need `else` and `return` goes outside |
| `elif [expression]:`    | Add an extra expression |
| `else:`                 | If other branches are not a match |
| `import module as m`    | Rename a module |
| `from module import a`  | Import items from module |
